1. 同步删除的文件及目录这个选择只有在作业类型是同步类型的时候才可能。在选定了这个选项后，当执行同步任务时，如果源数据删除了文件或者目录，则删除的文件及目录也将同步到目的地。
2. 压缩数据这个选项只有在作业的类型是 ZIP 时才可用。当选择这个选项时，把包的数据将被压缩。不选择这个选项的原因是为了快速解压。
3. 加密数据这个选项只有在作业的类型是 ZIP 时才可用。当前可用的加密算法是 AES128、AES192、AES256，后缀的数字越大，则加密的强度超强。但是你设置的密码才是最关键的钥匙。
4. 执行指定次数后合并这个选项只有在任务不是同步时才可用。这个选项的作用是帮助用户自动化管理数据。当执行的任务达到指定的合并次数后，对于不同的任务执行方法将有区别：
  • 对于复制作业，在执行完当前的复制操作后，将进行合并操作。合并操作的过程是将之前所有复制的数据重新复制到以当前时间字串为名字的另一个目录中。当你的文件系统支持硬连接特性，那么这个合并过程基本无法感知。比如 NTFS 文件系统就支持这样的特性。如果你的文件系统不支持这样的特性，那么合成过程则视复制数据的大小和多少。
  • 对于 ZIP 作业，在执行完当前的打包操作后，将通过分析开成一个最新的快照树，将这这个快照树中对应的文件从对应的 ZIP 文件中恢复到一个新的 ZIP 文件。这个过程视文件的大小、多少以及数据加密压缩情况会有不同的过程。
5. 档案分割尺寸这个选项只针对 ZIP 类型的作业类型。当 ZIP 单文件超过指定的大小时，将被分割成指定大小的文件。
6. 档案注释当你需要对作业的执行指定注释时使用。
1. 包含与排除过滤选项只能支持一种过滤方式，要么是排除，要么是包含。它们刚是互无交叉的过滤选项。
2. 文件与目录过滤选项即可以针对整个数据源下文件及目录，或者作业所包括的数据源的某个子目录。如果指定的子目录不在某个数据源下，则只会增加分析的时间，不会起到相应的作用。并且指定目录时，只针对这一些目录，而不会应用到此目录的下级子目录。
1. 启动信息后执行即指定的作业将在系统启动后执行。
2. 关闭系统时执行即指定的作业将在系统关闭时执行，无论本软件是否在运行。
3. 在指定的某个时间执行即在某个指定的时间运行。运行时间的指定可以按以星期或者以日期的方式。当指定的执行计划已经过期时，将不再执行。执行计划的最高的分辨率是一个小时。
